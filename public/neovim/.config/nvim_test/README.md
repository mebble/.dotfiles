# Test neovim config

This is a minimal neovim config for testing, debugging, etc.

Run:

```
NVIM_APPNAME=nvim_test nvim
```

Netrw file explorer:

```
# Open:
:Ex
# New folder:
:!mkdir path/rel/to/nvim_test
# New file in current netrw dir:
% (while in netrw)
```
